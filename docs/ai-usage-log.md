# AI 활용 내역 로그

> NAN 2026 사전과제 제출물 4번(AI 활용 기술 문서 PDF)의 원본 자료.
> 개발 중 AI 도구를 사용할 때마다 아래 형식으로 그때그때 추가할 것. 제출 직전에
> 몰아서 복원하려 하면 반드시 누락되니, 커밋할 때 같이 기록하는 습관을 들인다.

## 기록 형식

```
### YYYY-MM-DD — [작업 항목]
- 사용 도구: (예: Claude Code, ChatGPT, Midjourney, Stable Diffusion + ControlNet 등)
- 목적: (예: AutoJump 스크립트 초안 작성 / 캐릭터 컨셉 아트 생성 / 발판 수묵 변환 프롬프트 설계)
- 주요 프롬프트/지시:
  > (실제 사용한 프롬프트 원문 또는 핵심 지시사항 요약)
- 결과물: (커밋 링크 또는 파일 경로)
- 사람의 수정/검토 내용: (AI 결과물을 그대로 썼는지, 얼마나 고쳤는지)
```
